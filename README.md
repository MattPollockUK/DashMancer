# DashMancer – Data Analytics Dashboard

DashMancer is an interactive data portfolio dashboard built using Streamlit.

It combines multiple analytics projects into one unified interface where users can explore datasets, key metrics, and visualisations.

## Projects Included

### Superstore Sales Analysis
Explores sales performance across regions and product categories.

Key insights:
- West region generated the highest sales and profit
- Over 10,000 orders analysed

### Toyota Used Car Market Analysis
Analyses UK used Toyota car data with a focus on hybrid adoption and pricing trends.

Key insights:
- Hybrid vehicles account for ~30% of listings
- Clear price variation by fuel type

### Recipe Traffic Prediction
A machine learning project predicting high-traffic recipes.

Key insights:
- Logistic regression achieved strong recall for identifying high-traffic recipes
- Category analysis showed potatoes dominating high-traffic recipes

## Dashboard Features

- Interactive project navigation
- Key metrics for each dataset
- Data previews
- Visual analytics charts

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Run Locally

Clone the repository

```bash
git clone https://github.com/MattPollockUK/DashMancer.git
cd DashMancer
```

```

Run the Streamlit dashboard

```bash
streamlit run app.py
```

Once the server starts, open the local URL shown in the terminal (usually http://localhost:8501) to view the DashMancer dashboard in your browser.
