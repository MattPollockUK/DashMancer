# 🚀 DashMancer – Interactive Data Analytics Dashboard

> A unified dashboard for exploring multiple real-world data analysis projects and uncovering actionable insights.

**Live Demo (Local)**: Run the app locally via Terminal (instructions below)

## 🚀 Run Locally
1. Clone the repository:
git clone https://github.com/MattPollockUK/DashMancer.git
cd DashMancer
2. Install dependencies:
pip install -r requirements.txt
3. Launch the Streamlit dashboard:
streamlit run app.py
4. Open the local URL shown in the terminal (usually http://localhost:8501) to view DashMancer.

## 📊 Overview
DashMancer is an interactive data portfolio built with Streamlit that combines multiple analytics projects into one clean, user-friendly interface. It allows users to explore datasets dynamically, view key metrics and insights, and interact with visualisations in real time. This dashboard demonstrates practical applications of data analysis, visualization, and machine learning, all in a single, easy-to-use interface.

## 💡 Why this matters
Many businesses and individuals struggle to understand their data quickly, combine multiple analyses in one place, and extract actionable insights without technical tools. DashMancer shows how Python dashboards can solve these problems efficiently, turning raw data into meaningful insights.

## 📁 Projects Included
### 🛒 Superstore Sales Analysis
- Analysed over 10,000 orders  
- Identified West region as highest in sales and profit  
- Explored performance across regions and categories  

### 🚗 Toyota Used Car Market Analysis
- Analysed UK used car listings  
- Found hybrid vehicles make up ~30% of listings  
- Identified pricing differences by fuel type  

### 🍽️ Recipe Traffic Prediction (Machine Learning)
- Built a logistic regression model  
- Achieved strong recall for predicting high-traffic recipes  
- Identified potatoes as a dominant category  

## ⚙️ Dashboard Features
- Interactive project navigation  
- Key metrics for each dataset  
- Data previews  
- Visual analytics charts  

## 🧰 Tech Stack
